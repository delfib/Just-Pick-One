# Just Pick One

For the chronically indecisive. The universe will choose for you.

A fun, zero-dependency web app that takes your question and a list of options, and randomly picks one for you — with cosmic flair.

---

## How to run this project

No installation. No dependencies. No server needed.

Just open `index.html` in your browser.

```bash
# Clone the repo
git clone https://github.com/delfib/Just-Pick-One

# Open in browser
open index.html
```
